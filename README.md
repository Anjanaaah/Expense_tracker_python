# Expense_tracker_python

A Python-based Expense Tracker that allows users to manage and monitor their expenses through a menu-driven interface. The application uses SQLite for persistent data storage and implements complete CRUD operations for expense management.

## 🚀 Overview

Managing daily expenses manually can be difficult and error-prone. This project provides a simple solution for recording, updating, tracking, and analyzing expenses using Python and SQLite.

## ✨ Features

- Add new expenses
- View all expenses
- Update expense details
- Delete expense records
- Calculate total expenses
- Persistent data storage using SQLite
- Menu-driven user interface

## 🛠️ Tech Stack

- Python
- SQLite
- Git & GitHub

## 🗄️ Database Design

The application uses a relational database with the following fields:

| Field | Type |
|---------|---------|
| id | Integer (Primary Key) |
| amount | Real |
| category | Text |
| description | Text |
| date | Text |

## 🔧 Core Functionalities

### Expense Creation
Users can record expenses with amount, category, description, and date.

### Expense Tracking
All recorded expenses can be viewed in a structured format.

### Expense Modification
Existing expense records can be updated whenever required.

### Expense Deletion
Unwanted or incorrect records can be removed.

### Expense Analysis
The system calculates total expenditure from stored records.
